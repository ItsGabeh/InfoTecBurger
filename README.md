# InfoTecBurger

Pequeño programa de ejemplo en java Swing con conexion a base de datos sql

### Prerrequisitos

Necesitas tener instalado lo siguiente:

*   **JDK (Java Development Kit):** De preferencia Versión 25.
*   **Un IDE compatible com Maven (NetBeans o IntelliJIdea)**

### Instalación y Apertura en NetBeans

Sigue estos pasos para abrir el proyecto en NetBeans:

1.  **Clonar el Repositorio:**
    Abre tu terminal (o la interfaz de Git) y descarga el código:
    ```bash
    git clone --depth=1 https://github.com/ItsGabeh/InfoTecBurger.git
    ```

    O descarga el .zip y extraelo en una carpeta que reconozcas

2.  **Abrir el Proyecto en NetBeans:**
    *   Abre NetBeans IDE.
    *   Ve a **File** -> **Open Project**.
    *   Navega hasta la carpeta donde clonaste el proyecto.
    *   Selecciona la carpeta raíz del proyecto (donde se encuentra el archivo `pom.xml`) y haz clic en **Open Project**. NetBeans reconocerá automáticamente que es un proyecto Maven.
    *   Cambia las credenciales de la base de datos en el UserController para que pruebes el proyecto
